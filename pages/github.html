vamos con github



Github es una especie de red social de programadores. Con este sitio podemos subir nuestros proyectos y lograr que otras personas colaboren en ellos.


Primero cramos cuenta github

Ve a https://github.com/ (está en inglés). Haz clic en “sign up”.

Llena el formulario: Nombre de usuario. Dirección de e-mail (es recomendable usar el mismo email que usaste anteriormente en tu perfil de Git anteriormente). Contraseña.

Luego “Next: select a plan”, para seleccionar el plan que queremos tener.
